My Works

Movie Recommendation System

Problem: Users often face "choice paralysis" from vast movie libraries, making it difficult to discover new titles tailored to their specific tastes.

Action: I built a content-based filtering model using Python (Pandas, Scikit-learn) to analyze movie metadata from the TMDB 5000 dataset. I then deployed this model as a beautiful, interactive web application using Streamlit and the TMDB API to fetch live movie posters.

Result: The finished app provides users with 20 instant, relevant movie recommendations based on their selection, enhancing content discovery and demonstrating skills in machine learning, API integration, and front-end development.
